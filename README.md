# Sistema Comercial – Tkinter

Aplicação desktop com 3 módulos:
- Cálculo de comissões
- Controle de estoque
- Cálculo de juros

## Estrutura
- `data/` → dados fixos
- `services/` → regras de negócio
- `ui/` → interface Tkinter
- `main.py` → inicia a aplicação

## Rodar o projeto
```bash
python src/main.py
